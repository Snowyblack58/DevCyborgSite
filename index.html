Dev Cyborg

An assistant thingy in messenger

This is just messenger,
But it has developer commands...
!hello
Bot
hi
!note John "Hey, don't forget to push!"
Bot
I will notify John next time he is active.

Features

---

!hello

Bot says hi back

!hello
Bot
hi

!ping <user>

Bot returns a user object

!ping John
Bot
{"name":"John Smith","firstName":"John","vanity":"john.smith","thumbSrc":"<url>","profileUrl":"https://www.facebook.com/John.Smith","gender":2,"type":"user","isFriend":false,"isBirthday":false}

!help

Bot tells you all its commands

!help
Bot
Hal the Dev Cyborg Help:
!help
!task add <task>
!task edit <index> <new>
!task del <index>
!task list
!news <amount (min 5, max 30)>

!task add <task>

Bot adds your task to the tasks list

!task add "Include access token in second request"
Bot
Include access token in second request was added to tasks

!task list

Bot prints all your tasks

!task list
Bot
0 : github integration
1 : syntax highlighting

!task edit <index> <new>

Bot updates a task at the index

!task edit 1 "New task"
Bot
Task 1 is now New Task

!task del <index>

Bot deletes the task at that index

!task del 1
Bot
Task 1 has been deleted

!news <num>

Bot gets that number of articles from hacker news

!news 6
Bot
Software Library: Amiga https://archive.org/details/softwarelibrary_amiga&tab=collection Another Metroid 2 Remake v1.0 http://metroid2remake.blogspot.com/ Google Outpaces Facebook on Getting India Connected to Internet http://www.bloomberg.com/news/articles/2016-08-05/google-outpaces-facebook-on-getting-india-connected-to-internet How to become a data scientist https://www.experfy.com/blog/how-to-become-a-data-scientist-part-1-3 Steam in Docker https://hub.docker.com/r/andrey01/steam Apple should stop selling four-year-old computers http://www.theverge.com/2016/8/4/12373776/2012-macbook-pro-still-alive-not-dead-why

Technologies

---

  • Node.JS was used as the main language for developing Hal and the Bot.JS framework.

  • Amazon Web Services was used for the hosting of this website before it was not used for the hosting of this website.

  • We used a third party Facebook Messenger API, developed by Schmavery.
